Privacy Policy for Layout Guard

Effective date: May 27, 2026

Layout Guard is a browser extension that plays a sound when text typed on a web page appears to use the wrong keyboard layout.

Data Collection

Layout Guard does not collect, sell, share, or transmit personal information or typed content. It has no accounts, analytics, telemetry, advertising, backend service, or remote API integration.

Local Text Processing

While the extension is enabled, text in supported editable fields on ordinary web pages is checked locally against bundled keyboard mappings and local dictionaries. This processing is used only to decide whether to play a warning sound. Typed text is not stored by the extension or sent anywhere.

Stored Settings

Layout Guard uses chrome.storage.sync to remember user preferences, such as whether it is enabled, detection mode, minimum word length, cooldown, volume, enabled layouts, interface language, and excluded sites. These settings do not include typed text.

Website Access

The extension runs on web pages in order to observe typing in supported editable fields. It does not have access to the browser address bar, New Tab search field, or native Ctrl+F find bar. Users can disable detection or exclude specific sites in settings.

Network Requests And Remote Code

All runtime code, dictionaries, keyboard mappings, and generated sound behavior are packaged with the extension. Layout Guard does not make third-party network requests and does not load or execute remote code.

Changes

This policy may be updated when the extension changes. An updated policy will be published at this same URL with a new effective date.
